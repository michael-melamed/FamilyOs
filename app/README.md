# 📁 app/

## מה זה | What this is
תיקיית הבסיס של Next.js App Router המכילה את כל הדפים והניתובים של האפליקציה. | The base Next.js App Router directory containing all application pages and routes.

## מה יש כאן | Contents
- `layout.tsx` — Root layout containing RTL configuration and global providers
- `page.tsx` — Main landing page
- `globals.css` — Global stylesheet with Tailwind imports
- `(auth)/` — Authentication routes group
- `dashboard/` — Dashboard route

## תלויות | Depends on
Next.js core modules, global CSS

## מי תלוי בזה | Used by
Next.js Framework Router
